# metil_editor

metil_editor

a tool to create and edit `metil_mesh`s

## usage

```
metil_editor [?path_to_import_and_or_export]
```

## keybindings

| keybinding | action |
|-|-|
| `tab` | toggle movement mode (camera|cursor) |
| `period` | add vertex |
| `slash` | add index (index value of closest vertex index) |
| `command`+`s` | export file (exports to first parameter provided to `metil_editor`) |
| `g`+`[xyz]` | toggle grid axis visibility |
